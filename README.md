## 🚀 デプロイ情報

### 現在の公開URL
- **GenSpark Space**: `https://iviqvppm.gensparkspace.com/`

### カスタムドメイン設定手順

#### 1. ドメイン側のDNS設定
お持ちのドメインのDNS設定で以下を追加：

**Aレコード（推奨）**
```
Type: A
Name: @ または サブドメイン（例: recruit）
Value: [GenSparkが提供するIPアドレス]
TTL: 3600
```

**CNAMEレコード（サブドメインの場合）**
```
Type: CNAME
Name: recruit（例）
Value: iviqvppm.gensparkspace.com
TTL: 3600
```

#### 2. GenSpark側の設定
1. GenSparkのPublishタブを開く
2. 「Custom Domain」セクションでドメイン名を入力
3. DNS設定が反映されるまで待機（最大48時間）

#### 3. SSL証明書（HTTPS）
- GenSparkが自動的にLet's Encrypt証明書を発行
- 設定完了後、自動的にHTTPSでアクセス可能
